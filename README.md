# EXAMPLE: EMPTY MENU BAR IN MACOS WITH SWIFTUI

This is a simple example of how to build a macOS menu bar app using SwiftUI with a popover.


https://github.com/user-attachments/assets/6daf893f-9950-4d1a-9fe0-59a8353bdafc


## How to Run

1. Clone this repository.
2. Open the project in Xcode.
3. Build and run the project. The app will appear in the macOS menu bar with Globe icon.
4. Click on Globe icon in menu bar to toggle the popover.
5. And enjoy with "Click me" button to increment counter.

## Files

- [`EmptyMenuBarApp.swift`](EmptyMenuBar/EmptyMenuBarApp.swift): Sets up the menu bar app and handles the popover.
- [`PopoverContentView.swift`](EmptyMenuBar/PopoverContentView.swift): Defines the content of the popover.

## Requirements

- macOS 15+
- Xcode 16+
